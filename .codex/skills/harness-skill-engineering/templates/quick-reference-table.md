# Quick Reference 表格模板

Quick Reference 是 `SKILL.md` 的路由表，用低 Token 指引 Agent 加载正确资源。

## 通用模板

```md
| 用户意图/关键词 | 加载资源 | 用途 |
|---|---|---|
| <关键词、口语、行业术语> | `reference/<file>.md` | <获取公式、规范、判断标准或背景> |
| <需要固定格式> | `templates/<file>.md` | <生成统一输出> |
| <需要确定性计算或扫描> | `scripts/<file>` | <执行可复现逻辑> |
```

## 示例

```md
| 用户意图/关键词 | 加载资源 | 用途 |
|---|---|---|
| 合同风险、违约责任、付款条款 | `reference/contract-risk.md` | 获取风险分类和审查标准 |
| 要审查报告、出具意见 | `templates/review-report.md` | 生成统一审查报告 |
| 批量提取金额和日期 | `scripts/extract_terms.py` | 执行确定性字段提取 |
```

## 写法要求

- 每行只指向真正需要的资源。
- 关键词使用用户真实表达，而不只用内部术语。
- 用途要说明“读完获得什么”，不要只写“参考”。
- 不要把长解释写进表格。
